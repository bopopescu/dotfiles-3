
ログインシェルはbashでtmux,screen起動後はzshに切り替えて使用しています。

Install
::

  install.sh


Python
::

  .startup/python27

vim 7.3
::

  # installed vim 7.3
  ~/.startup/vim73
  # install vundle.git
  git clone http://github.com/gmarik/vundle.git ~/.vim/bundle/vundle
  # bundle install
  vim -c "BundleInstall"


Uninstall
::

  uninstall.sh
